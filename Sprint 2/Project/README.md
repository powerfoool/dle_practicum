# Проект модуля. Нейросеть для автодополнения текстов
Курс Яндекс Практикума "Инженер по глубокому обучению нейросетей"

Проект реализует систему автодополнения текста с использованием двух подходов: рекуррентной нейронной сети (LSTM) и предобученного трансформера (DistilGPT‑2). Проводится сравнение качества работы моделей на общем датасете.

## Результаты
В ходе проекта решена учебная задача по 3/4 текста твита предсказания оставшейся 1/4 этого сообщения. Для решения использовались LSTM обученная в рамках проекта и предобученный трансформер DistilGPT‑2.
LSTM на валидационной выборке показала качество ROUGE1 = 0.0395, ROUGE2 = 0.0033, тогда как DistilGPT‑2 показал качество ROUGE1 = 0.0583, ROUGE2 = 0.0057.
